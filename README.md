# OTSGE Draft

## How do I install these formulae?

`brew install otsge/draft/<formula>`

Or `brew tap otsge/draft` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "otsge/draft"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
